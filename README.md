# Flora Vision

A responsive plant landing page built with React, Vite, and Tailwind CSS.

## Features

- Responsive landing page
- Hero Section
- Trending Plants Section
- Top Selling Plants Section
- Modern UI
- Tailwind CSS

## Tech Stack

- React.js
- Vite
- Tailwind CSS

## Installation

Clone the repository

```bash
git clone https://github.com/vinaybansode/flora-vision.git
```

Go to the project folder

```bash
cd flora-vision
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
